# Contribuindo

Este repositório centraliza a documentação estratégica, técnica e operacional do Clube da Régua.

## Como Contribuir

1. Crie uma branch a partir de `main`.
2. Faça alterações objetivas e bem estruturadas.
3. Use títulos claros e linguagem simples.
4. Abra um pull request descrevendo o motivo da mudança.

## Padrões de Escrita

- Use português brasileiro.
- Seja direto, moderno e próximo da realidade do barbeiro.
- Evite excesso de linguagem corporativa.
- Prefira exemplos práticos.

## Commits

Use mensagens no formato:

```text
tipo: resumo da alteração
```

Exemplos:

- `docs: atualiza roadmap`
- `brand: adiciona regras de logo`
- `arch: detalha multitenancy`
