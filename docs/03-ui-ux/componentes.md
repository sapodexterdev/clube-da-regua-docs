# Componentes

## Botões

- Primário: fundo dourado, texto preto, alto contraste.
- Secundário: fundo grafite, texto branco, borda cinza.
- Destrutivo: vermelho moderado, confirmação clara.

## Cards

Cards devem ser escuros, objetivos e fáceis de escanear.

## Inputs

Campos com fundo grafite, labels claras e validações diretas.

## Chips

Usados para filtros, status e seleção de pagamento.

## Status

- Novo: dourado.
- Confirmado: verde.
- Cancelado: vermelho.
- Contatado: azul discreto.
