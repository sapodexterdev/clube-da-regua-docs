# Design System

## Princípios

- Rápido.
- Simples.
- Elegante.
- Tecnológico.
- Urbano.

## Cores

- Preto Principal: `#0D0D0D`.
- Grafite: `#1A1A1A`.
- Dourado: `#F2C14E`.
- Branco: `#FFFFFF`.
- Cinza: `#7A7A7A`.

## Tipografia

- Títulos: Bebas Neue ou Anton.
- Texto e interface: Inter.

## Superfícies

- Fundo principal escuro.
- Cards em grafite.
- Ações primárias em dourado.
- Estados selecionados em dourado com texto preto.

## Ícones

Ícones minimalistas, com traço simples, evitando excesso visual.
