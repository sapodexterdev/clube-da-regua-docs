# Identidade Visual

## Direção Visual

O Clube da Régua deve parecer moderno, urbano, direto e confiável.

## Elementos

- Tema escuro como base.
- Dourado para ações, destaques e estados selecionados.
- Grafite para cards, menus e superfícies.
- Cinza para textos secundários.
- Imagens reais de barbearias, pessoas e ambientes urbanos.

## Evitar

- Visual genérico de banco de imagens.
- Paleta laranja predominante.
- Excesso de gradientes.
- Linguagem visual corporativa demais.

## Estilo de Imagem

Usar barbearias reais, luz baixa, concreto, metal, grafite, rua, streetwear, detalhes de navalha e bastidores.
