# Lançamento

## Estratégia

Lançar com foco em donos de barbearias que já sentem dor de agenda, caixa e crescimento.

## Fases

1. Pré-lançamento com conteúdo educativo.
2. Lista de espera para barbearias fundadoras.
3. Demonstrações guiadas.
4. Oferta para primeiros clientes.
5. Coleta de depoimentos e casos de sucesso.

## Canais

- Instagram.
- WhatsApp.
- Indicação.
- Comunidades locais.
- Parcerias com fornecedores do setor.
