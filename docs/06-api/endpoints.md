# Endpoints

## Supabase REST

Os endpoints são expostos automaticamente por tabela.

## Principais Recursos

- `barber_shops`
- `barbers`
- `services`
- `booking_requests`
- `appointments`
- `payments`
- `cash_movements`

## Convenções

- Autenticação via Bearer token.
- Chave pública apenas para operações permitidas por RLS.
- Inserções públicas restritas a solicitações de agendamento.
