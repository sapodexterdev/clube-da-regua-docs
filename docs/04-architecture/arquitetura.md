# Arquitetura

O Clube da Régua usa arquitetura de monorepo com apps Flutter e backend Supabase.

## Camadas

- Apps Flutter.
- Pacote compartilhado.
- Supabase Auth.
- Supabase Postgres.
- Supabase Storage.
- Row Level Security.
- Deploy web via Vercel.

## Princípios

- Simplicidade operacional.
- Segurança por tenant.
- Evolução modular.
- Separação entre app cliente e app gestão.
