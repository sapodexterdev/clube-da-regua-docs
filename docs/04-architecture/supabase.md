# Supabase

## Responsabilidades

- Autenticação.
- Banco Postgres.
- Storage de mídia.
- Regras de acesso com RLS.
- APIs REST automáticas.

## Integrações

Os apps usam REST com `SUPABASE_URL` e `SUPABASE_ANON_KEY`.

## Segurança

Toda tabela sensível deve usar Row Level Security e políticas por barbearia.
