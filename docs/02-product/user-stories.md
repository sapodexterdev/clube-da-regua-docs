# User Stories

## Cliente

- Como cliente, quero escolher barbeiro, serviço, data e horário para solicitar um agendamento.
- Como cliente, quero informar WhatsApp para receber confirmação.
- Como cliente, quero escolher se vou pagar por PIX, dinheiro ou cartão.

## Barbeiro

- Como barbeiro, quero ver meus horários do dia.
- Como barbeiro, quero bloquear horários indisponíveis.
- Como barbeiro, quero acompanhar minha comissão.

## Dono

- Como dono, quero cadastrar barbeiros e serviços.
- Como dono, quero aprovar, cancelar ou converter solicitações.
- Como dono, quero ver caixa, estoque e faturamento.

## Régua IA

- Como gestor, quero receber sugestões de horários, campanhas e ações para melhorar a ocupação.
