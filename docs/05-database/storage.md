# Storage

## Buckets

- `barbershop-media`: imagens públicas de barbearias, barbeiros, serviços e campanhas.

## Tipos de Arquivo

- Logos.
- Fotos de barbeiros.
- Capas de barbearias.
- Imagens de serviços.
- Assets de campanha.

## Regras

- Validar tamanho e tipo de arquivo.
- Separar caminhos por `barber_shop_id`.
- Evitar exposição de documentos pessoais.
